Для ознайомлення

Внутрішні документи

Регламенти, положення та інші документи

  • Регламент надання послуг комісіонера на підставі генерального договору про надання інвестиційних послуг (Цей документ незабаром буде доступний для ознайомлення)

  • Регламент надання послуг комісіонеру в рамках здійснення брокерського обслуговування (Цей документ незабаром буде доступний для ознайомлення)

  • Положення (політика) щодо конфлікту інтересів та особистих угод ТОВ «БТС Брокер» (Цей документ незабаром буде доступний для ознайомлення)

  • Положення (політика) з питань винагороди ТОВ «БТС Брокер» (Цей документ незабаром буде доступний для ознайомлення)

  • Положення про порядок розгляду скарг та інших звернень клієнтів, потенційних клієнтів та професійних учасників фондового ринку ТОВ «БТС Брокер» (Цей документ незабаром буде доступний для ознайомлення)

  • Внутрішнє положення депозитарної установи (Цей документ незабаром буде доступний для ознайомлення)

  • Повідомлення про обробку персональних даних (завантажити | переглянути)

  • Відомості для клієнта або потенційного клієнта згідно ст.7 ЗУ «Про фінансові послуги та фінансові компанії» (завантажити | переглянути)
